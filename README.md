# Helena · Mis 15 — Save the Date

Página "Save the Date" (simple, sin fotos) para los 15 de Helena. Hecha en **Next.js**, lista para subir a **Vercel** sin base de datos.

## Probar localmente

```bash
npm install
npm run dev
```

Abrí http://localhost:3000

## Editar los datos

Todo lo editable está en **`app/data.js`**: fecha, salón, valor de la tarjeta, alias, fecha límite y número de WhatsApp.

## Subir a Vercel

1. Subí esta carpeta a un repositorio de GitHub.
2. En [vercel.com](https://vercel.com) → **Add New → Project** → importá el repo.
3. Vercel detecta Next.js solo. Click en **Deploy**. Listo ✅

> Esta es la página corta. La invitación completa está en la carpeta `invitacion/` (repo aparte).
